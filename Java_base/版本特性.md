# Java版本特性（使用过的）

## LTS版本

***LTS***：Long-Term Support 长期支持版本

### 8（1.8）

- ### Lambda表达式

如，创建线程时

老版本写法：

```java
    Thread t7 = new Thread(new Runnable() {
        @Override
        public void run() {
            //TO-DO
        }
    });
```
将光标放在new Runnable()上；IDEA提示：

    Anonymous new Runnable() can be replaced with lambda
    
新版本写法：

```java
    Thread t8 = new Thread(() -> {
        //TO-DO
    });
```
优点：代码更简洁
