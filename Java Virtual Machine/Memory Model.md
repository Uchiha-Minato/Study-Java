# Java内存模型 - JMM, Java Memory Model

JMM 定义了一套在多线程读写共享数据（成员变量，数组）时，对数据的可见性、有序性和原子性的规则和保障。

## 原子性 - 一连串不可分割的操作

要么全部一起执行，要么都不执行，中间不能插入其他任何操作

保证原子性 - **synchronized 同步关键字(同步锁)**

语法：

    synchronized( 对象 ) {
        // 原子操作代码
    }

**同一时刻只能有一个线程进入大括号内执行原子操作**
